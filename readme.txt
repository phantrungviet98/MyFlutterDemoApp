# my_demo_flutter_app

Các bước cài đặt

- Cài đặt Flutter. Ta có thể tham khảo document từ Google để cài đặt https://flutter.dev/docs/get-started/install

- Sau khi cài đặt Flutter thành công ta mở Máy ảo Emulator Android hoặc Simulator IOS.

- flutter run tại thư mục project để chạy ứng dụng.

Lưu ý: Trong file lib/main.dart có chứa route để vào từng demo

Ta có thể thay initialRoute: FacadeScreen.routeName để vào Facade Demo
 hoặc initialRoute: SoldierGameScreen.routeName để vào Flyweight Demo
